Rhyme with me!
==============

1. 用的这位大佬的 [repo](https://github.com/chinese-poetry/chinese-poetry)
2. clone 下来, 指定个路径
3. 导入他的诗词库
   
    ```shell
    $ python init.py /path/to/chinese-poetry/json/
    ```
4. 可以玩了!
   
    ```shell
    $ python rhyme.py 这个真是幺蛾子
    ```

其实就很简单, 查找替换了而已, 不知道原文读起来也没气势, 先这样吧哈哈哈
